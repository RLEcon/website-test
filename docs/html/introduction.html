Introduction¶

written by Joao B. Sousa

Welcome to the test website for RL Economics.

You can clone this repo and add new content, and publish it by simply pushing the changes into the online repo. This will automatically publish the new html files in the website hosted by github.

To add content, create a new .rst file in the source folder, and add this file to the index.rst.

Then, navigate to the root folder and run make html to build.

For more details follow the offical Getting started tutorial.
